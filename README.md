# Space Educational Website

A comprehensive educational website about space, combining contributions from multiple team members.

## Overview

This website covers various topics in astronomy and space science, including:

- **Galaxies**: Types of galaxies and the Milky Way
- **Black Holes**: What they are, types, and event horizons
- **Earth**: Our home planet and its unique features
- **Nebulae**: Cosmic clouds where stars are born
- **Outer Planets**: Jupiter, Saturn, Uranus, and Neptune
- **Sun & Stars**: The Sun and stellar phenomena

## Structure

- `index.html`: Main homepage with navigation to all topics
- `galaxy.html`: Galaxies page
- `black-holes.html`: Black holes page
- `earth.html`: Earth page
- `nebula.html`: Nebulae page
- `outer-planets.html`: Outer planets page
- `sun-and-stars.html`: Sun & Stars page
- `black-holes/`, `earth/`, `outer-planets/`, `sun-and-stars/`: asset folders used by the section pages

## How to Run

1. Ensure you have Python installed
2. Navigate to the project directory
3. Run: `python -m http.server 8000`
4. Open your browser to `http://localhost:8000`
